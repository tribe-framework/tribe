# tribe-core

## To update to latest composer for tribe-core

```
docker compose exec php_tribe composer update --no-interaction --optimize-autoloader
docker compose exec php_junction composer update --no-interaction --optimize-autoloader
docker compose exec php_dist composer update --no-interaction --optimize-autoloader
```

---

## Tribe - a web project management framework.

Tribe is a project management framework that allows us to build platforms, products, applications and web interfaces, using a modular approach. It provides a coherent language for UX, content, design and code collaborations.
